우선 실행 하기 위해 

1. npm install

2.npx react-native run-android (안드 기준) 
  npx react-native run-ios(iOS 기준) 
  실행을 한번 부탁 드립니다. 

실행시에 에러가 잡힌다면 아마 2가지 경우가 있을건데 한개는 안드로이드 에뮬레이터 미 설치, 
또 하나는 실행 잡기전에 yarn 기준으로 작업된거 중에 충돌이 있을수도 있으니 노드 모듈 폴더 내 파일 삭제 후
yarn install로 재 설치 부탁드립니다. 
참고로 자바 문제로 충돌이 있긴 하나, 핸들러 부분이니 이거는 무시 해도 됩니다. 

"projectConfig": {
    "backendApiBaseUrl": "해당 백엔드 접속할 IP넣으세요"
  }

여기에 보시면 해당 백엔드 연결할 호스트 IP연동해야 합니다. 안그러면 서버 연결이 안됩니다. 

3. 백엔드 파일을 따로 올리니 백엔드 파일을 받으면 똑같은 메모장이 있습니다. 
메모장대로 따로 설치후 구동을 해야 합니다.
*만약 다운로드를 하지 않았다면 반드시 다운을 하셔야 하고,
이후 백엔드 메모를 확인하시면 됩니다. 

4.문제가 되는 부분은 리액트 네이티브 방에 스샷이나 영상을 올려 주시면 확인 후 조취 취하겠습니다.
